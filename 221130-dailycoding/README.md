튜플이란 무엇인가?

셀수있는 수량의 순서있는 열거 또는 어떤 순서를 따르는 요소들의 모음을 튜플이라고 한다.

1. 중복된 원소가 있을 수 있다.
2. 정해진순러가 있으며, 원소의 순서가 다르면 서로 다른 튜플이다.
3. 튜플의 원소개수는 유한하다.

어떻게 풀것인가? 

입출력 예제 2,1,3,4 이라는 튜블이 있을 때 이 튜블에 속하는 원소는 가장 앞에 있는 숫자부터 차례대로 붙여나가자
2, {2,1} , {2,1,3}, {2,1,3,4}
