# 이해 
구명보트 구출 근데 보트는 작아서 2명밖에 못타고 무게제한이 있다.

구명보트를 최대한 적게 이용해서 사람들을 구하려고 한다.

사람들의 몸무게는 배열로 주어진다.
70 50 80 50 키로의 사람들이 있고 100키로 제한이라면
50, 50 2명이 탈 수 있고
나머지 경우의 수는 사람들이 탈 수 가 없다. 

모든 경우의 수를 봐야 할 것 같다.(2중 for문 사용해야할듯하다)

구명보트의 무게제한은 항상 구출자의 몸무게보다 크다.
아무도 구하지 못할 가능성은 없다.

핵심은 최대한 많은 사람을 구할 수 있는 경우를 찾는 것인데
